# HTMLスライドをPDF/Googleスライドに変換する方法

## 📄 PDFへの変換

### 方法1: 自動スクリプト（推奨）

作成したスクリプトを使用します：

```bash
cd notes
./html-to-pdf.sh slide-自治会.html slide-自治会.pdf
```

または、プロジェクトルートから：

```bash
./notes/html-to-pdf.sh notes/slide-自治会.html notes/slide-自治会.pdf
```

**注意**: ChromeまたはChromiumが必要です。

### 方法2: ブラウザの印刷機能（手動）

1. HTMLファイルをブラウザで開く
   - `notes/slide-自治会.html` をダブルクリック
   - または、ブラウザで `file://` パスを開く

2. ブラウザの印刷機能を使用
   - **Mac**: `Cmd + P`
   - **Windows**: `Ctrl + P`

3. 設定を調整
   - **送信先**: 「PDFに保存」を選択
   - **レイアウト**: 横向き（Landscape）を推奨
   - **用紙サイズ**: A4 または カスタム（1280x720px相当）
   - **余白**: なし
   - **背景のグラフィック**: オン（重要！）

4. PDFとして保存

### 方法3: オンラインツール

- [HTML to PDF](https://www.html-pdf.com/)
- [PDF24](https://tools.pdf24.org/ja/html-to-pdf)

HTMLファイルと画像ファイルをアップロードして変換します。

## 📊 Googleスライドへの変換

**注意**: PDFからGoogleスライドへの直接変換は、レイアウトによってはうまく動作しない場合があります。以下の方法を推奨します。

### 方法1: 各スライドを画像としてエクスポート（推奨・最も確実）

#### 自動化スクリプトを使用（Node.jsが必要）

1. **Node.jsとPuppeteerをインストール**
   ```bash
   # Node.jsがインストールされているか確認
   node --version
   
   # プロジェクトにPuppeteerをインストール
   cd notes
   npm init -y
   npm install puppeteer
   ```

2. **スクリプトを実行**
   ```bash
   node html-to-slides.js slide-自治会.html slide-images
   ```

3. **Googleスライドに画像を追加**
   - 新しいGoogleスライドを作成
   - 「挿入」→「画像」→「アップロード」で各画像を追加
   - 各画像を1つのスライドとして配置
   - 画像を右クリック→「背景に設定」でスライド全体に表示

#### 手動でスクリーンショット

1. **HTMLを開く**
   - ブラウザで `notes/slide-自治会.html` を開く

2. **各スライドをスクリーンショット**
   - 各スライドを個別にスクリーンショット
   - **Mac**: `Cmd + Shift + 4` で範囲選択してスライドを選択
   - **Windows**: `Win + Shift + S` で範囲選択
   - ファイル名を `slide-01.png`, `slide-02.png` のように保存

3. **Googleスライドに挿入**
   - 新しいGoogleスライドを作成
   - 各画像を「挿入」→「画像」→「アップロード」で追加
   - 画像を右クリック→「背景に設定」でスライド全体に表示
   - または、画像をドラッグしてスライド全体をカバーするように調整

### 方法2: PDF経由（動作しない場合あり）

1. **PDFを生成**
   - 上記の方法でPDFを作成

2. **Googleドライブにアップロード**
   - Googleドライブを開く
   - PDFファイルをアップロード

3. **Googleスライドに変換を試行**
   - PDFファイルを右クリック
   - 「アプリで開く」→「Googleスライド」を選択
   - **注意**: この方法はレイアウトが崩れる場合があります

4. **動作しない場合**
   - 方法1（画像エクスポート）を使用してください

## 💡 ヒント

### PDFの品質を向上させる

- **フォント**: ブラウザでHTMLを開いてからPDF化すると、Webフォントが正しく表示されます
- **画像**: 画像のパスが正しいことを確認（`../images/`）
- **サイズ**: スライドサイズ（1280x720px）に合わせてPDFサイズを調整

### Googleスライドでの編集

- PDFから変換した場合、テキストは編集可能な場合とそうでない場合があります
- 画像として挿入した場合は、テキストを再入力する必要があります
- レイアウトは自動調整されるため、必要に応じて手動で調整してください

## 🔧 トラブルシューティング

### 画像が表示されない

- 画像のパスが正しいか確認（`../images/`）
- 画像ファイルが `images/` フォルダに存在するか確認

### PDFのレイアウトが崩れる

- ブラウザの印刷設定で「背景のグラフィック」をオンにする
- 用紙サイズをカスタム（1280x720px）に設定
- 余白を「なし」に設定

### スクリプトが動作しない

- ChromeまたはChromiumがインストールされているか確認
- スクリプトに実行権限があるか確認（`chmod +x html-to-pdf.sh`）
