# 画像フォルダ

このフォルダには、Markdown記事で使用する画像ファイルを保存します。

## 画像の保存方法

1. 画像ファイル（.jpg, .png, .gif, .webp など）をこのフォルダに保存
2. Markdownファイルから相対パスで参照

## ファイル命名規則（推奨）

- わかりやすい名前を付ける
- 日付や記事名を含める
- 例: `season2-ep3-qrcode.png`, `2024-01-15-coffee-machine.jpg`

## 画像の参照方法

Markdownファイルから以下のように参照します：

```markdown
![画像の説明](images/画像ファイル名.jpg)
```

例：
```markdown
![QRコードの例](images/qrcode-example.png)
```
